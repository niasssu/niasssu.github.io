journey through linux

it started with just trying to find a way to fix my slow laptop.

ubuntu
it starts with an ubuntu partition taking up about half of the laptop's hardrive. this solved the computer's performance problems for a while, but still i had no idea what was going on behind the gui. so as i continued doing as i normally did and eventually ubuntu became just as slow as windows. well, even now it's still faster, but it's not much of an experienced inprovement anymore. i had just built another tower pc with my dad and he put windows on it. i immediately looked for a comfortable linux and found mint.

mint
i installed mint and immediately look to figure out what i did wrong. well, not that quickly. i still played around and downloaded apps i didn't need, but i quickly learned about the linux terminal, finally. huh, that was easy-- ONE MISTAKE AND IT'S ALL OVER.

oh, oh yeah that's right. nevermind.
i really messed with the terminal if i absolutely had to, but over about a few years i figured out how to be safer with it. i did try doing one of the joke commands on the ubuntu laptop [specifically "sudo rm -rfv / --no-preserve-root", very stupid: DO NOT TRY] and i cancelled it midway through because i just wanted to see if it would actually go through AND IT DID. i suffered for about 5mins just watching in horror/shock as i saw all my files getting deleted. it's something i think all linux users have done at least once.

after that i went right back to mint as ubuntu became almost unusable. mint is still the os i use on my main computer, but i have tried a few others out of curiosity.

manjaro
the laggiest operating system i have EVER used, might be worse than windows even. also it keeps wanting to update. i liked the design and all, it is just it takes a while for the operating system to get going. other than that, i say stop pressuring me to update and it's ok.

easyOS
i found this one on distrowatch one day, it advertised itself as a lightweight OS that could run entirely off of USB. i wanted to try it out, to see just how good it was.

it's extremely simple and easy to set up, though i say it is heavily dependent on the mouse in the GUI. it feels nice to use, i like it! it's amazing, that all of it can run entirely off my USB drive.

tails
looked for more OSs that could run off the USB stick, and this one i found quite interesting. i wanted to try it out.

a bit slow to start but has a nice feel. it does its work just as advertised.
